
== Processing Audio Visualization (PAV)

PAV is a music visualization project based on Processing/Java.

 * <tt>libpav</tt> is the visualization library
 * <tt>pav</tt> is an application that visualizes music streamed to it over the network using <tt>libpav</tt>
 * <tt>pav-player</tt> is a simple audio player with tag support that can stream audio data to <tt>pav</tt> over the network

PAV is released under the GPL v3 license.
